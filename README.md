# Local Tube
Selfhosted media stream service that allow to download video from Internet and watch it from own server.

## Technologies
- .NET Core 
- Entity Framework 3.1
- Fluent API
- AutoMapper
- Swagger
- SQL Server

### SPA
- Angular 9
- Ng-Bootstrap
- Ngx-Toastr

### Mobile-desktop app
- MAUI
