# Local Tube
Selfhosted media stream service that allow to download video from Internet and watch it from own server.

## Technologies
- .NET Core 
- MS SQL Server
- EF Core
- Quartz
- Telegram
- Torrent-tracker
- Yt-DLP
- PhantomJS
- SignalR
- Fluent API
- AutoMapper
- Swagger


### SPA
- Angular 
- Ng-Bootstrap
- Ngx-Toastr

### Mobile-desktop app
- MAUI 
- SignalR
